# Artefak Keluaran

Folder ini menyimpan hasil akhir yang layak dibagikan. File sementara, log, dan hasil inspeksi berukuran besar tidak masuk Git.

## PDF

`pdf/laporan_analisis_kemiskinan_indonesia_2015_2026.pdf` adalah laporan 14 halaman yang menjelaskan tren, kesenjangan spasial, faktor terkait, model, validasi, dan proyeksi.

## Spreadsheet

- `spreadsheets/poverty_data_foundation_2015_2025/`: fondasi data dan pemeriksaan cakupan.
- `spreadsheets/poverty_analysis_2015_2026/`: analisis, benchmark, prediksi, dan dashboard seed.

File `*.inspect.ndjson` adalah artefak pemeriksaan lokal dan dikecualikan melalui `.gitignore`.
